# 새로 배운 내용
- Host Computer: 내가 현재 쓰고 있는 컴퓨터
- Guest Computer: 가상 컴퓨터
- Repository: 저장소
- PHP 주석: //주석, /* 여러 줄 주석 */ 
- 주석 설정 및 해제 단축키: Ctrl + /
- PHP에서 문자열에 "나 '를 직접 출력하고 싶으면 앞에 \\(백슬래쉬) 입력
