# 새로 배운 내용
- SSH: 네트워크 상의 다른 컴퓨터에 로그인하거나 원격 시스템에서 명령을 실행할 수 있는 프로토콜
- SSH 설치 방법
　
- Git: 원격 저장소
　
- VSCode와 Github의 연동
　
- header()를 이용하여 원하는 페이지로 바로 이동 가능
- 너무 많은 데이터를 한꺼번에 검색하는 경우 Allowed memory size(메모리 소진)
