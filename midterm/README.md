# 개발 환경 소개
1. 데이터베이스: MariaDB


데이터베이스를 MySQL을 선택하지 않고 MariaDB를 선택한 이유는 MariaDB에는 MySQL보다 더 많은 기능을 가지고 포함하고 있고, 성능면에서도 더 가볍고 빠르다고 알고 있어 MariaDB를 선택하게 되었다. 또한, 라이선스 문제에서 MySQL은 다수에게 공식적인 목적으로 서비스를 제공한다면 라이선스를 구입해야하지만 MariaDB는 라이선스를 구입할 필요가 없다. 

2. 서버 사이드 언어(백엔드): PHP

3. 클라이언트 사이드 언어(프론트엔드): HTML, CSS

4. 웹 서버 : Linux, Apache Web Server
